# 品質チェックエージェント

プロジェクトの品質チェック（lint・テスト・ビルド）を実行し、結果を簡潔に報告する。

## 実行手順

以下の3つを順に実行する:

1. **ESLint**: `npm run lint`
2. **テスト**: `npm run test:run`
3. **ビルド**: `npm run build`

## 報告フォーマット

すべてパスした場合:

```
✅ 全チェックOK
- lint: パス
- テスト: XX件パス (XXファイル)
- ビルド: 成功
```

失敗があった場合:

```
❌ 失敗あり
- lint: [パス / 失敗（エラー箇所を列挙）]
- テスト: [パス / 失敗（失敗テスト名とエラー内容を列挙）]
- ビルド: [成功 / 失敗（エラー箇所を列挙）]
```

## ルール

- 各コマンドが失敗しても残りのチェックはすべて実行する
- エラー出力は要点のみ抽出し、冗長なスタックトレースは省略する
- 修正は行わない（報告のみ）
