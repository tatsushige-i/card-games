# マージ後クリーンアップスキル

PR がマージされた後、ローカルの feature ブランチを削除して main ブランチを最新化する。

## 前提条件

- PR がすでにマージ済みであること
- 現在 feature ブランチにいること（main ブランチでは実行しない）

## 実行プロセス

### Step 1: 状態確認

1. 現在のブランチ名を取得する
2. `main` ブランチにいる場合は **エラーとして中止** し、feature ブランチで実行するよう案内する
3. 未コミットの変更がないことを確認する。変更がある場合は **エラーとして中止** し、先にコミットまたはスタッシュするよう案内する

### Step 2: クリーンアップ

1. `git switch main` で main ブランチに切り替える
2. `git branch -d <branch-name>` でローカルの feature ブランチを削除する
3. `git pull` で main ブランチを最新化する

### Step 3: 完了報告

削除したブランチ名と現在の状態をユーザーに報告する。
