# 概要

- ビデオポーカー（Jacks or Better）を実装する。

## 機能要件

- 1人用のビデオポーカーゲーム。
- 52枚のデッキから5枚のカードが配られる。
- プレイヤーはホールド（保持）するカードをタップで選択できる。
- 「ドロー」ボタンで、ホールドしていないカードを新しいカードに交換する。
- 最終手札でポーカーの役を判定する。
- 10ラウンド制。各ラウンドで役に応じたポイントを獲得する。
- 中間ラウンド（1〜9）は結果表示後、ダイアログなしで自動的に次のラウンドを開始する。
- 10ラウンド終了後に合計スコアでゲーム終了ダイアログを表示する。

## 役と配当（ポイント）

| 役 | ポイント |
|----|---------|
| ロイヤルフラッシュ | 800 |
| ストレートフラッシュ | 50 |
| フォーカード | 25 |
| フルハウス | 9 |
| フラッシュ | 6 |
| ストレート | 4 |
| スリーカード | 3 |
| ツーペア | 2 |
| ジャックスオアベター（J/Q/K/Aのワンペア） | 1 |
| ノーハンド | 0 |

## ベストスコア

- 合計スコア（10ラウンドの総得点）の最高記録を保存する。

## UI要件

- 5枚のカードを横一列に表示する。
- カードタップでホールド状態をトグルする。ホールド中のカードは視覚的に区別する（上に持ち上げる + 「HOLD」ラベル表示）。
- 配当表（ペイテーブル）を画面上部に表示し、現在の役をハイライトする。
- ラウンド結果表示時に役名を大きく表示する。
